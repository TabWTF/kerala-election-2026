Step-by-Step Deployment

1

Create a GitHub Account

Go to github.com and sign up with your email. It's free and takes 2 minutes.

2

Create a New Repository

Once logged in, click the + icon → New repository. Name it kerala-election-2026. Set it to Public. Click "Create repository".

3

Upload the index.html File

In your new repo, click "uploading an existing file". Drag and drop the index.html file from your outputs folder. Write a commit message like "Initial launch" and click Commit changes.

4

Create a Free Vercel Account

Go to vercel.com → Sign up with GitHub. This links Vercel to your GitHub account automatically.

5

Import Your Repository

In Vercel dashboard, click Add New → Project. You'll see your GitHub repos listed. Click Import next to kerala-election-2026. Leave all settings as default and click Deploy.

🚀 Deploying... ✓ Done! Your site is live.
6

Your Site is Live!

Vercel gives you a URL like kerala-election-2026.vercel.app. Share it anywhere — WhatsApp, Twitter, Instagram. Every time you update the index.html on GitHub, Vercel auto-redeploys in ~30 seconds.

7

Update Candidate Data Daily

As parties announce 2026 candidates, open index.html in any text editor (Notepad on Windows, TextEdit on Mac). Find the CANDIDATES_2026 section and add candidate names. Re-upload to GitHub and it goes live automatically.

// Example — add a candidate: 133: [ // Nemom {name:"O. Rajagopal", party:"BJP", alliance:"NDA"}, {name:"K. Muraleedharan", party:"INC", alliance:"UDF"}, ],

Pro Tip: Share Individual Constituency Cards
