Step-by-Step Deployment

1

Create a GitHub Account

Go to github.com and sign up. Free and takes 2 minutes.

2

Create a New Repository

Click + → New repository. Name it kerala-election-2026. Set to Public. Click Create.

3

Upload index.html

In your repo, click "uploading an existing file". Drag and drop the index.html from your outputs folder. Write "Initial launch" and click Commit changes.

4

Create a Free Vercel Account

Go to vercel.com → Sign up with GitHub. This links both accounts automatically.

5

Import and Deploy

In Vercel, click Add New → Project. Find kerala-election-2026 and click Import. Leave all settings default and click Deploy.

🚀 Deploying... ✓ Done! Your site is live.
6

Your Site is Live!

You get a URL like kerala-election-2026.vercel.app. Share it everywhere. Every time you update index.html on GitHub, Vercel auto-redeploys in ~30 seconds.

7

Update Candidate Data Daily

Open index.html in any text editor (Notepad / TextEdit). Find the CANDIDATES_2026 section and add new candidate entries. Re-upload to GitHub to go live instantly.

// Example: add 2026 candidates for Dharmadom (id 12) 12: [ {name:"Pinarayi Vijayan", party:"CPI(M)", alliance:"LDF", incumbent:true}, {name:"T. Siddarth", party:"INC", alliance:"UDF"}, ],
